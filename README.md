# Student Performance Analyzer

## Description
This project analyzes student performance using Python and generates insights such as average marks, grades, and top performers.

## Features
- Calculates average marks
- Assigns grades based on performance
- Identifies top-performing student
- Visualizes data using bar charts
- Saves processed data to CSV

## Technologies Used
- Python
- Pandas
- Matplotlib

## Output
- Displays student performance analysis
- Generates graphs for visualization
- Saves results as output.csv

## How to Run
1. Open the .ipynb file in Google Colab
2. Run all cells
3. View output and graphs
